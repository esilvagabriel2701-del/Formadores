Glossário de Programação

Prof. Gabriel

A

Algoritmo

Conjunto de instruções ordenadas para resolver um problema ou executar uma tarefa.

API (Application Programming Interface)

Conjunto de regras que permite que diferentes sistemas ou programas se comuniquem.

Array

Estrutura de dados que armazena vários elementos do mesmo tipo em uma sequência.

B

Backend

Parte da aplicação responsável pela lógica de negócios e processamento de dados.

Bug

Erro ou falha em um programa que produz comportamento inesperado.

Banco de Dados

Sistema utilizado para armazenar, organizar e gerenciar informações.

C

Classe

Modelo utilizado na programação orientada a objetos para criar objetos.

Compilador

Programa que traduz código-fonte para linguagem de máquina ou executável.

Código-fonte

Conjunto de instruções escritas por um programador em uma linguagem de programação.

D

Debug

Processo de identificar e corrigir erros em um programa.

Deploy

Ato de disponibilizar uma aplicação para uso em produção.

Dependência

Biblioteca ou pacote externo utilizado por um projeto.

E

Estrutura de Dados

Forma de organizar e armazenar dados para facilitar seu acesso e manipulação.

Exceção (Exception)

Evento que interrompe o fluxo normal de execução de um programa.

F

Framework

Conjunto de ferramentas e bibliotecas que facilita o desenvolvimento de aplicações.

Frontend

Parte visual da aplicação com a qual o usuário interage.

G

Git

Sistema de controle de versão utilizado para gerenciar alterações em projetos.

GitHub

Plataforma para hospedagem e colaboração em projetos que utilizam Git.

H

HTML

Linguagem de marcação utilizada para estruturar páginas web.

I

IDE

Ambiente de desenvolvimento integrado que reúne ferramentas para programação.

Instância

Objeto criado a partir de uma classe.

J

JSON

Formato leve para troca de dados entre sistemas.

L

Loop

Estrutura que permite repetir um bloco de código várias vezes.

M

Método

Função associada a uma classe ou objeto.

O

Objeto

Instância de uma classe que possui atributos e métodos.

Open Source

Software cujo código-fonte pode ser estudado, modificado e distribuído livremente.

P

Parâmetro

Valor recebido por uma função ou método.

Ponteiro

Variável que armazena o endereço de memória de outra variável.

R

Repositório

Local onde o código-fonte de um projeto é armazenado e versionado.

S

Sintaxe

Conjunto de regras que define como escrever comandos em uma linguagem.

Stack Overflow

Comunidade online onde programadores compartilham conhecimento e solucionam dúvidas.

V

Variável

Espaço nomeado na memória utilizado para armazenar dados.

Versionamento

Controle das alterações realizadas em arquivos e projetos ao longo do tempo.
