Programação Front-End

Prof. Gabriel

Front-end é a parte de um site ou aplicação com a qual o usuário interage diretamente. Ele envolve a criação da interface visual, incluindo textos, imagens, botões e menus. As principais tecnologias utilizadas são HTML, CSS e JavaScript. O objetivo do front-end é proporcionar uma experiência agradável, intuitiva e responsiva para os usuários em diferentes dispositivos.
